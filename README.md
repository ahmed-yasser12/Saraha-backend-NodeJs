# Saraha-backend
A RESTful backend API built with Node.js, Express.js, and MongoDB. Features include user authentication, email verification, password recovery, JWT authorization, file uploads, and secure message management following a scalable modular architecture.

🚀 Features

- User Authentication & Authorization (JWT)
- Email Verification
- Password Reset & OTP
- File Uploads with Multer
- MongoDB & Mongoose Integration
- Request Validation
- Error Handling Middleware
- Modular Project Structure
- Secure Password Hashing with bcrypt
- RESTful API Design
